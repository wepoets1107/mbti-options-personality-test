# MBTI 期权人格测试

一个把 MBTI 人格和期权策略做成趣味测试的小网页。

16 道简单选择题，测出你的交易人格，并匹配一张期权人格卡。每张卡包含人格关键词、对应策略和 payoff 示意图，风格是赛博金融漫画海报。

## 在线体验

部署完成后可通过 GitHub Pages 访问。

## 内容

- 16 型 MBTI 人格卡
- 16 个不重复的期权策略
- 本地静态网页，无需后端
- SignalPlus 交易入口
- 冰火岛社区打赏信息

## 本地预览

在项目目录运行：

```bash
python -m http.server 8787 --bind 127.0.0.1
```

然后打开：

```text
http://127.0.0.1:8787/
```
